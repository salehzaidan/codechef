# CodeChef

My CodeChef problem solutions.

## Prerequisites

- GCC 11 (`gcc`)
- Google Test (`libgtest-dev`)
- CMake (`cmake`)
- GDB (`gdb`) (Optional, for debugging only)

## Usage

1. Build all problems:

```sh
cmake --build ./build
```

2. Run a specific problem:

```sh
./build/AVGPROBLEM
```

## Debugging

Run the problem with GDB:

```sh
gdb ./build/AVGPROBLEM
```
